# Module 9. Authentication & Security

## Description

Implementing secure authentication and protecting routes in Next.js applications.

## Skills

- Auth providers (NextAuth or custom)
- OAuth 2.0 (Google, GitHub)
- JWT & refresh tokens
- Session handling
- Role-based access control (RBAC)
- Protected routes and middleware
- Secure API routes
- Password hashing (bcrypt, argon2)
- CSRF / XSS / CORS protection
- Rate limiting & brute-force protection
- Environment variables & secrets management

## Topics

- Authentication in App Router
- Middleware-based auth
- Server Actions + auth
- Cookies vs headers
- Secure redirects

## Example Projects

- Login and registration flow
- OAuth login (Google/GitHub)
- Dashboard with protected access
- Role-based dashboard (admin / user)
- API with JWT and protected endpoints
- Middleware-based route protection
